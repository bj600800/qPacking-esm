# qPacking
Exploring hydrophobic stacking patterns in protein folding superfamilies.

**Dependance:**
```
python=3.12
conda install conda-forge::biotite (version=1.0.1)
pip install tqdm
conda install dssp (version=4.4.8)
pip install melodia-py (version=0.1.4)
```

**Tools:**
```
conda install -c conda-forge -c bioconda mmseqs2
conda install -c bioconda seqkit
conda install bioconda::cd-hit
pip install pdb-tools
```
